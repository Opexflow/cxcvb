# cxcvb-frontend

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## For production version

```bash
npm run build
# or
yarn build
```

## Then start project

```bash
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
